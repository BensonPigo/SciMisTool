====設定為 Windows 服務====

使用NSSM：管理 Windows 服務的開源工具，全名為 "Non-Sucking Service Manager"

1. 下載 NSSM

下載最新版本的 NSSM（選擇適合的版本，通常是 64 位元版本）。
解壓縮下載的 ZIP 檔案，將 nssm.exe 放到一個方便的位置，例如 C:\Tools\NSSM。



2. 使用 NSSM 安裝服務
假設你想將一個應用程式（例如 myapp.exe）安裝為服務：

(1) 打開命令提示字元（CMD）或 PowerShell。
(2) 切到myapp.exe所在目錄
(3) 執行 .\nssm install MyServiceName
(4) 會彈出 NSSM 的GUI：
Path：輸入應用程式的完整路徑（例如 C:\MyApp\myapp.exe）。
Startup directory：輸入應用程式的工作目錄（例如 C:\MyApp）。
Arguments：如果需要傳遞參數，可以在這裡輸入。

(5) 點擊「Install service」按鈕，服務就會被安裝。



3. 安裝完成後，你可以使用以下指令管理服務：

啟動服務：
.\nssm start MyServiceName

編輯服務(使用GUI)：
.\nssm edit MyServiceName

停止服務：
.\nssm stop MyServiceName

重啟服務：
.\nssm restart MyServiceName

刪除服務：
.\nssm remove MyServiceName

也可以透過 Windows 的「服務」管理工具（services.msc）來查看和管理這個服務。
